# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-07-14

### Added

- Initial release of Masaj Vip website.
- Created responsive static website structure.
- Added SEO optimized landing pages for regions and cities.
- Added multilingual URL structure:
  - Turkish (`/tr/`)
  - English (`/en/`)
  - Arabic (`/ar/`)
- Added language selection dropdown with automatic language detection.
- Added dynamic language switching based on current URL.
- Added mobile responsive header behavior.
- Added compact mobile header animation on scroll.
- Added SEO optimized meta tags for all regional pages.
- Added canonical URLs for individual pages.
- Added Open Graph and Twitter Card metadata.
- Added favicon and theme color configuration.
- Added Google Fonts integration.
- Added structured CSS architecture:
  - Reset styles
  - Variables
  - Typography
  - Components
  - Responsive styles
- Added sitemap.xml for search engine indexing.
- Added robots.txt configuration.
- Added localized region pages for İstanbul, İzmir, Ankara, Antalya, Bursa and other supported locations.
- Added FAQ, About, Listings and Regions sections.

### Improved

- Improved mobile user experience.
- Improved page structure for SEO compatibility.
- Improved navigation usability with language selection.
- Improved header performance by optimizing scroll animations.
- Improved asset organization.

### Technical

- Built as a lightweight static HTML/CSS/JavaScript website.
- Implemented SEO-focused URL structure.
- Optimized resource loading and frontend organization.
